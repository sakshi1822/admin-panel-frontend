This is the Admin Panel for the Foodies web application.
It allows administrators to manage categories, products, orders, users, and overall system data through a clean and responsive dashboard.

Built using React.js, styled with modern UI components, and integrated with the backend through REST APIs.

Features

Dashboard with quick stats

Manage categories

Manage food items/products

Manage orders

Manage users

API integration with backend

Fully responsive UI
